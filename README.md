 ## 🚀 Getting Started

Follow these steps to get the SocialEcho project up and running on your local machine.

---

### 📁 Step 1: Clone the Repository

Clone the project from GitHub:

```bash
git clone  https://github.com/AshrithaAkula19/socilecho.git
cd socilecho
cd SocialEcho-main
```
📦 Step 2: Install Dependencies
Install required packages for both frontend and backend:

👉 Client (Frontend)
```bash
cd client
npm install
```
👉 Server (Backend)
```bash
cd ../server
npm install
```
🔐 Step 3: Setup Environment Variables
Create a .env file in both client and server directories.
Use .env.example files (already present) as a guide to set the environment variables:

For email services

For authentication

For content moderation APIs

▶️ Step 4: Start the Application
Start Backend Server
```bash 
cd server
npm start
```
Start Frontend Client (in a new terminal)
```bash
cd client
npm start
```
